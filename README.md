## Future Features
- Dark Mode
- training programs
```
1. Signature Workout Templates
Pre-built workout programs inspired by famous fighters:

"GSP's Gymnastic Flow" - gymnastics-based strength work
"Demetrious Johnson's Cardio Circuit" - high-intensity cardio routine
"Khabib's Wrestling Grind" - wrestling-focused session
"Gordon Ryan's BJJ Drilling" - specific positional drilling sequences
"Buakaw's Muay Thai Basics" - traditional Muay Thai pad work
"Saenchai's Flow Sparring" - technical flow drills

Each template would include:

Duration and intensity level
Round structure (e.g., "5x5 minute rounds")
Specific techniques/exercises to focus on
Rest intervals
Notes on the athlete's training philosophy

4. Daily "Train Like" Challenges
Random daily workouts:

"Today: Train like Tony Ferguson" - unconventional workout (tire flips, wing chun dummy, etc.)
"Today: Train like Ngannou" - pure power session
Shows you a specific workout they're known for

Simple Version (MVP):

Dropdown menu of 10-15 famous athletes
Select one and see their "signature workout"
Log that you completed it (tracks as a special workout type)
Leaderboard showing which athlete's workouts you've done most
```
